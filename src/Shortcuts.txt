1. psvm -> main method
2. sout -> System.out.println();
3. souf -> System.out.printf();
4. Ctrl + D - дублиране на реда, на който се намира курсура на мишката
5. Ctrl + alt + L -> подравняване на кода
6. Shift + F6 -> Refactor/Rename - преименуване
7. Ctrl + alt + S -> отваряне на настройките
8. Alt + Enter -> Import, # quick fix
9. Ctrl + Shift + / -> закоментира маркирания код
10. Shift + F6 (може да изисква комбинация с Fn) -> промяна името на променлива на всички места където се среща в класа
